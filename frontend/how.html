How to use Zacs Web Films

How to add a movie:

Navigate to '/films' in the root folder.

Create a new file in '/films' with the name of the film. Make sure not to use spaces!

Inside the file can you add the movie as a .mp4 file. If you want a costome cover you can add a .jpg file. To add a description add a .txt file and write your description in that file. All files in your movie folder can be named whatever you like, just make sure they are in the correct format. (.mp4, .jpg, .txt)

How to start backend:

Before starting the backend of ZWF make sure you have node.js installed. Open the terminal in the root folder and run the following command: 'node ./zwf.js' To close the backend you can close the terminal or use ctrl+c.
